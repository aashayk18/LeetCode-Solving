# Intuition
Recursively swap the right and left branches of the tree.

# Approach
Check to see if the root of the tree exists, and if it does then recursively call the function on the left and right branches of the tree and assign those values to the right and left branches respectively. This way, all the branches and sub-branches of the tree are swapped i.e. the left becomes the right and vice-versa.
