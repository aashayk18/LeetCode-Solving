# Intuition
Get the difference of the unique elements present in the range `1 to n`, and the unique elements present in the given array.

# Approach
Use the `set()` method on the list `nums` to get all the unique elements present in it. Use the `set()` method to get the unique numbers in the range `1 to n`, here `n` being given by the length of the list nums. Subtract the first value from the second to find the missing numbers.
