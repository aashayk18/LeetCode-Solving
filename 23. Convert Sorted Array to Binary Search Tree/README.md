# Intuition
Recursively split the array and convert the array elements into binary search tree nodes.

# Approach
Set the condition for empty array to return None. Assign the middle element of the array as the root for sorting the array. Recursively perform the operation of converting the array elements into nodes of a tree using the TreeNode function, on the first half of the now split array and also the second half.
