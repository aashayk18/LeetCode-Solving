# Intuition
Check if the target exists in the array, if so return its index. If not, append the target to the array, sort the array, and return the index of the target.

# Approach
Run a for loop over the array, and check if any element matches the target. If so, return the index of that element. If not, append the target to the array, sort the array, and then return the index of the target in the array.
