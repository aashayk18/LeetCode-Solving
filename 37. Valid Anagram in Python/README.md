# Intuition
Check the counts of elements in both strings to see if they're equal or not.

# Approach
Use the `collections.Counter()` method to create a dictionary of the distinct characters of a string as the key and its total count in the string as the value. Do this for both the input strings, asnd check if these two dictionaries are equal. Return the boolean value for this expression.
