# Intuition
Check if the capitalized, uppercase and lowercase forms of the string match the current form.

# Approach
Use the `capitaliize()` method to get the proper sentence case form of the word. Check it against the current word to see if they match. Do the same with the uppercase and lowercase forms of the word using the `upper()` and `lower()` methods respectively. Return the boolean value for these checks.

# Complexity
- Time complexity:
$$O(N)$$

- Space complexity:
$$O(1)$$
